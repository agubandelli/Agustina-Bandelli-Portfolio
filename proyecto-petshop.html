Pet Shop

Desarrollo de una tienda online de productos para mascotas utilizando HTML, CSS y JavaScript. El sitio permite buscar productos, explorar un catálogo dinámico y ver el detalle de cada artículo, mostrando información como nombre, precio e imagen. Implementé la carga de datos desde arrays en JavaScript y cuidé la estructura visual y la experiencia de usuario, buscando un diseño limpio, funcional y fácil de navegar.

Herramientas utilizadas:

HTML CSS JavaScript Illustrator
